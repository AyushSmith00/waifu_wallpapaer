# waifu_wallpaper

A responsive anime wallpaper gallery built with React and Tailwind CSS. The application fetches wallpapers from the Waifu.im API and allows users to browse wallpapers by category and control the number of images displayed.

## Features

* Browse wallpapers by category
* Select the number of wallpapers displayed per page
* Custom React Hook for data fetching
* Responsive user interface with Tailwind CSS
* Dynamic rendering using React state and props

## Tech Stack

* React
* JavaScript
* Tailwind CSS
* Waifu.im API

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/waifu_wallpaper.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
